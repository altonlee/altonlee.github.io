My first coding language was C#, back when I tried to create a video game using the Unity engine back in the summer of my junior year of high school. I found C# to be incredibly hard to use, mostly because I had not started with the basics and also because I solely relied on YouTube video comments for learning, and gave up on my simple maze game after a month of development. Fast forward to second semester. I got to revisit the C language in my EE160 class, and although it was simple and easy to pick up, I absolutely hated the UI and how pointers and headers worked (the C IDE had small font that seemed impossible to change, so that sucked). 

Learning JavaScript was a nice breath of fresh air. I had just learned Java last year, so learning JavaScript was pretty easy to learn since a lot of principles are carried over from Java, aside from accessing an object’s properties via bracket notation, and not needing to initialize any arrays. I also love the fact that all variable types, like ``int`` and ``String``, have all been combined into a single ``var`` type (though I can see some problems with variable “conversions” later in the future…) Other than that, the transition to JavaScript has been very smooth. I think it is a very good software development language, since it is very easy to pick up and has a lot of real world applications (SoundCloud was developed on JavaScript!). 

The athletic software engineering teaching style in this class is a good concept, but personally I don’t really like it. I like learning in class and doing work at home, since that’s the learning style that I am accustomed to, but this style does give me incentive to research and learn more about the topic since there’s a lot more resources outside of the online lectures. The WODs are great and I do enjoy doing them, since it applies all of my knowledge of JavaScript and allows me to practice my quick thinking and problem solving skills in a timed manner. Overall, this class is my second most stressful class this semester (311 takes the cake on this one), but if I manage my time right, everything will be alright.
